## Requisitos

Se debe tener las siguientes herramientas instaladas:

- Python 3.9 >=
- modulo python venv

## Uso

Instalar las dependencias con los siguiente comandos:

crear el entorno virtual (por unica vez)
```bash
python3 -m venv .venv
```

activar el entorno virtual
```bash
source .venv/bin/activate
```

instalar las dependencias (si las hubiese)
```bash
pip install -r requirements.txt
```

si se quisiese desactivar el entorno virtual:

```bash
deactivate
```

Para ejecutar el programa, se debe correr el siguiente comando:

```bash
python3 server.py
```

Listo, ahora el programa deberia estar corriendo en un servidor local.
